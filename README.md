# 🦷 Projeto Clínica Odontológica  
### Landing Page Responsiva e Moderna

---

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **Landing Page moderna, elegante e totalmente responsiva** para uma clínica odontológica fictícia.

A aplicação foi desenvolvida utilizando **HTML**, **CSS** e **JavaScript**, com foco em performance, organização visual e experiência do usuário.

### 🎯 Objetivos do Projeto

- Criar uma interface visualmente profissional  
- Garantir responsividade para todos os dispositivos  
- Implementar interatividade com JavaScript  
- Simular um ambiente real de divulgação de serviços odontológicos  

---

## 🌐 Funcionalidades Implementadas

✔ Layout moderno e organizado  
✔ Design 100% responsivo  
✔ Menu adaptado para dispositivos móveis (menu hambúrguer)  
✔ Modo Escuro (Dark Mode)  
✔ Sistema de visualização de clima por cidade  
✔ Integração com API de CEP  
✔ Formulário de contato funcional  

---

## 🧩 Visão Geral das Seções

- **Home** → Seção principal com destaque visual (Hero Section)  
- **Serviços** → Apresentação dos principais tratamentos oferecidos  
- **Sobre** → Informações institucionais da clínica  
- **Contato** → Formulário para envio de mensagens  
- **Clima** → Consulta de temperatura da cidade informada  
- **Dark Mode** → Alternância entre tema claro e escuro  

---

## ⚙ Pré-requisitos

Para executar o projeto localmente, você precisará de:

- 🌍 **Navegador moderno** (Google Chrome, Firefox, Edge, etc.)
- 💻 **Editor de código** (recomendado: Visual Studio Code)
- 🔧 **Git** (opcional, para clonar o repositório)

---

## 🛠 Tecnologias Utilizadas

- **HTML5** → Estruturação semântica da página  
- **CSS3** → Estilização moderna e layout responsivo  
- **JavaScript** → Interatividade e manipulação do DOM  
- **Google Fonts** → Tipografia personalizada  
- **API de Clima** → Exibição dinâmica da temperatura  
- **API de CEP** → Consulta automática de localização via CEP
